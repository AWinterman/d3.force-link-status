Initialize the Status object with three parameters which describe the graph:

  • loops: true if a link can lead from a node to itself
  • directed: True if a link from node A to node B is the same a s a link form B to A
  • multiedge: True if you can have multiple links between a given pair of nodes.

Status#is checks for equality between two links, paying attention to whether the graph is directed. Links are equal if they point to the same objects.
